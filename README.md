## 🛠️ كيف تستعمله
1. ادخل للمتصفح على سوق كروم ابحث عن c3cfbstet استخرج كوكي 
2. اذهب الى ملف سروس **Cookie.txt** الصق كوكي
3. شغل البوت

4.اذ حدث لك مشكل لا تترد عندي اجي فيس
